行送りトークンは、行ボックスの高さの指定に使います。

CSSでは`line-height`を指しています。

## セマンティクストークン

| トークン名  | 値 | 使用例 |
| :--- | :--- | :--- |
| `NONE`    | 1    | [ラベルテキスト](/products/design-tokens/typography/#h3-4)、[Heading](/products/components/heading/) |
| `TIGHT`   | 1.25 | |
| `NORMAL`  | 1.5  | [段落テキスト](/products/design-tokens/typography/#h3-3) |
| `RELAXED` | 1.75 | |

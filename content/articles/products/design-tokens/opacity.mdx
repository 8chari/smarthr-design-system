不透明度トークンは、要素の不透明度の指定に使用します。

不透明度はCSSの`opacity`プロパティと同義です。`0%`で完全に透過し、`100%`で設定された要素の裏にある内容が完全に隠れます。

要素のデフォルトは`100%`です。

## プリミティブトークン

| トークン名 | 値（%） |
| :--------- | :------ |
| `0`          | 0%      |
| `25`         | 25%     |
| `50`         | 50%     |
| `75`         | 75%     |
| `100`        | 100%    |

## セマンティクストークン

| トークン名 | 値（%） | 利用例           |
| :----------- | :------ | :------------ |
| `ACTIVE`     | 100%    | 通常時         |
| `HOVER`      | 75%     | マウスオーバー時 |

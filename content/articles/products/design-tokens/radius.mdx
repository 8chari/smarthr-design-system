角丸トークンは、要素の境界の外側の角を丸めるために使用します。

CSSでは`border-radius`を指しています。

## 基準値
ベースの基準サイズは`16px`です。

## プリミティブトークン

| トークン名 | 値（rem） | 値（px） |
| :--- | :--- | :--- |
| `0`     | 0        | 0    |
| `0.25`  | 0.25rem  | 4px  |
| `0.375` | 0.375rem | 6px  |
| `0.5`   | 0.5rem   | 8px  |
| `1`     | 1rem     | 16px |
| `625`   | 625rem   | 10000px |

## セマンティクストークン

| トークン名 | 値（rem） | 値（px） |
| :--- | :--- | :--- |
| `S`    | 0.25rem  | 4px  |
| `M`    | 0.375rem | 6px  |
| `L`    | 0.5rem   | 8px  |
| `FULL` | 625rem   | 10000px |
